浏览器下

JS中的this关键字由函数的调用者决定，谁调用就this就指向哪个。如果找不到调用者，浏览器环境下this将指向windows对象。 con是装宝藏的盒子， this是盒子里的宝藏， foo是一把钥匙 ，谁用钥匙把盒子打开， 盒子里的宝藏就是谁的 

      var con = function (){
        console.log(this)
      }
      var obj = {
        foo: con 
      }
      obj.foo() // foo 就是钥匙
      
      var bar = obj.foo; // bar就是钥匙
      bar() // window.bar() // window就是用钥匙的那个人

      var person = {
        p1: {
          caozep: {
            me: con 
          }
        }
      }
      person.p1.caozep.me()// me就是钥匙， caozep就是用钥匙的那个人
